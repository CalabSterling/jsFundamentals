Talk to the hand

Instructions

You are going to create a html unordered list to describe things about you based on the hand

  • Pointer Finger: someone you look up to
  • Middle Finger: something that bothers you
  • Ring Finger: something you love to do
  • Pinky Finger: something you could do a little better
  • Thumb: something you would be doing right now if you were not in class
